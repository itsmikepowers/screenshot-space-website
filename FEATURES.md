# Screenshot Space

A lightweight macOS screenshot tool that lives in your menu bar. No account, no cloud, no bloat — just fast keyboard-driven screenshots.

## How It Works

Screenshot Space uses modifier keys as shortcuts. By default:

| Action | Shortcut | Description |
|--------|----------|-------------|
| **Full screen** | Tap **⌥ Option** | Captures entire screen instantly |
| **Drag selection** | Hold **⌥ Option** | Hold ~0.25s then drag to select a region |
| **Region recapture** | Tap **🌐 Fn** | Re-capture a previously defined region |

All screenshots are saved locally and copied to clipboard automatically.

## Features

### Instant Capture
- **Full screen** — Tap Option to capture your entire screen
- **Drag selection** — Hold Option, then drag to select any area
- **Region recapture** — Define a region once, recapture it with a single key

### Screenshot Gallery
- Browse all screenshots in a grid or list view
- Bulk select, copy, rename, and delete
- Drag and drop to Finder, browsers, or any app
- Double-click for full-size preview

### OCR Search
- Search by text inside your screenshots
- On-device processing with Apple Vision (no cloud)
- Highlighted match snippets in results

### Menu Bar Integration
- Quick access from the menu bar
- Recent screenshots with thumbnails
- One-click copy to clipboard

### Customizable Shortcuts
- Choose any modifier key: Control, Shift, Option, Command, or Fn
- Set tap or tap-and-hold triggers
- Adjust hold threshold (0.1s–1.0s)
- Enable/disable individual capture modes

### System Integration
- Launch at login
- Show/hide from Dock
- Custom screenshot folder
- Auto-updates

## Requirements

- macOS 13 or later
- Accessibility permission (for keyboard shortcuts)
- Screen Recording permission (for region capture only)

## Privacy

- All screenshots stay on your Mac
- OCR runs locally with Apple Vision
- No account required
- No data leaves your device
